# 🔥 ReconPulse — Web Reconnaissance & Vulnerability Insights Dashboard

<img width="612" height="408" alt="reconpulse" src="https://github.com/user-attachments/assets/dd54ed19-b46e-42d4-b416-2eaaf25894b1" />


ReconPulse is a modern, lightweight web reconnaissance framework designed for bug bounty hunters, penetration testers, and security researchers.

Built with simplicity and efficiency in mind, ReconPulse provides automated target analysis, JavaScript endpoint extraction, parameter discovery, and early-stage vulnerability detection — all within an interactive dashboard.

---

## 🚀 Features

- 🌐 Target Fetching & Response Analysis
- 📡 HTTP Header Inspection
- 📜 JavaScript File Extraction
- 🔍 Endpoint Discovery from JS
- 🧩 Parameter Identification
- 💀 Basic Vulnerability Detection:
  - Potential SSRF / Open Redirect
  - Potential IDOR
- 📊 Risk Scoring System
- 🖥️ Interactive Dashboard (Streamlit-based)

---

## ▶️ Usage

```
pip install requests beautifulsoup4 streamlit
streamlit run recon_pulse.py

# Open in browser:
http://localhost:8501

```
